### Installation
Clone repository.
``` bash
git clone https://github.com/bencoderus/clipboard-health.git
```

Install package.
``` bash
cd clipboard-health && npm install
```

Run function.
``` bash
npm run start
```

Run test.
``` bash
npm test
```
